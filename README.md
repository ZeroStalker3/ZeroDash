# 🖥️ ZeroDash

Асинхронный Telegram-бот на Python для удаленного мониторинга состояния операционной системы, администрирования домашней серверной инфраструктуры и управления хостами в локальной сети.
🚀 Основные возможности
Мониторинг системы: Отслеживание нагрузки на аппаратную часть сервера (CPU, RAM, Disk, температура) в реальном времени.
Управление процессами: Интеграция с ОС через выполнение системных команд и Bash-скриптов.
Сетевое управление: Возможность удаленного включения персонального компьютера в локальной сети (Wake-on-LAN).
Безопасность: Ограничение доступа к управлению сервером на уровне ID администратора (handlers/admin.py).
Docker-ready: Полная контейнеризация приложения для быстрой развертывания на целевом сервере.

🔧 Технологический стек
Язык программирования: Python 3.13+
Фреймворк: aiogram 3.x (Asyncio)
Среда развертывания: Docker, Docker Compose
Базовые утилиты: ОС Linux (Debian), Bash

🛠️ Настройка и конфигурация

Перед запуском необходимо настроить переменные окружения. Создайте файл .env в корневой директории проекта:
Фрагмент кода

BOT_TOKEN=your_telegram_bot_token
ADMIN_ID=your_telegram_user_id

📦 Варианты развертывания
Вариант 1: Развертывание через Docker Compose (Рекомендуемый)

Для изоляции приложения и запуска в бэкграунде на домашнем сервере выполните команду:
Bash

docker compose up -d --build

Вариант 2: Локальный запуск (Для разработки и отладки)
Клонируйте репозиторий:
git clone https://github.com/ZeroStalker3/ZeroDash.git
cd ZeroDash

Создайте и активируйте виртуальное окружение:
python -m venv .venv
source .venv/bin/activate  # Для Linux/macOS
# или
.venv\Scripts\activate     # Для Windows

Установите зависимости:
pip install -r requirements.txt

Запустите бота:
python main.py

📝 Логирование

Все ключевые события работы бота, ошибки выполнения Bash-команд и системные уведомления записываются в файл bot.log с указанием времени и уровня критичности (INFO, WARNING, ERROR).
